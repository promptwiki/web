# 기능 목록 정의

## AI와의 대화 요약

> "간단한 CRM을 만들고 싶은데, 필수 기능이 뭘까?"

AI가 제안한 필수 기능들 중에서 선별함.

## MVP 기능 목록

### Must Have (필수)
1. **고객 목록** - 이름, 이메일, 전화번호, 회사
2. **고객 상세** - 기본 정보 + 메모 + 활동 기록
3. **검색/필터** - 이름, 회사로 검색
4. **사용자 인증** - 이메일 로그인
5. **대시보드** - 최근 활동, 통계 요약

### Should Have (되면 좋음)
- 태그/라벨 시스템
- CSV 내보내기
- 이메일 알림

### Could Have (나중에)
- 팀 협업
- API 연동
- 모바일 앱

## 차별점

> "5분 안에 시작, 5분 안에 가치 얻기"

복잡한 설정 없이 바로 사용 가능한 것이 핵심.
